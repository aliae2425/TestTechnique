## Plateforme de Quiz en Ligne — Test Technique

Ce projet est une plateforme de quiz en ligne conçue pour l'évaluation de compétences techniques.

### Objectif
Permettre aux utilisateurs de passer des tests techniques sous forme de quiz interactifs, afin d'évaluer leurs connaissances dans différents domaines.

### Fonctionnalités principales
- Création et gestion de quiz techniques
- Interface utilisateur simple et intuitive
- Suivi des résultats et des scores

### Utilisation
1. Accédez à la plateforme
2. Sélectionnez un test technique
3. Répondez aux questions du quiz
4. Consultez vos résultats
